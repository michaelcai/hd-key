
# Chainstack.js

Javascript API for integration with Chainstack blockchains using Chainstack RPC API.

## Installing / Getting started

### NodeJS

```shell
npm install @tchainstack/chainstack.js

// or

yarn add -S @tchainstack/chainstack.js
```
